ZteerのUTAU関連物置き場
プラグイ ン
  • 連続音一括設定 version 1.60 更新 2015/11/3
　連続音音源で作業する際に必ず必要な設定を一括して行うプラグインです。プ ラグイン名は『連続音一括設定』ですが、単独音のみ使用する設定も可能です。


  • 拡張ピッチエディタ version 2.15 更新 2014/12/21
以前の名称は『Mode1用ピッチエディタ』でしたが、Mode2に対応した ため名称を変更しました。
選択された複数の音符を、専用のピアノロール上でまとめて編集することができます。
フリーハンドでピッチベンドを引く機能や、モジュレーションをピッチベンドに反映する機能があります。


  • 拡張エンベロープエディタ version 1.63 更新 2014/12/21
複数の音符のエンベロープをまとめて編集するエンベロープエディタです。
クロスフェード部分を保護したまま、エンベロープを調整することができます。


  • 拡張フラグエディタ version 1.13 更新 2015/09/05
複数の音符のフラグをまとめてGUI上で編集するフラグエディタです。
音量、モジュレーション、子音速度も同様の操作で編集することができます。


  • 周波数トレーサー version 2.42 更新 2014/12/14（2015/5/23不足ファイルの追加）
  • 音声ファイルのピッチや音量の変化を分析し、UTAU上に再現します。
  • 現在設定されている歌手で元音声を真似させる『トレースモード』と、元音声を音源化し、UTAU上に再現する『音声補正モード』があ ります。
  • 音符の区切り位置指定やピッチの補正等をGUIで行うことができます。
  • 連続音音源を使用している場合、自動的に連続音の歌詞に変換します。
  • 音量はエンベロープに適用されます。
  • 音量推定、キー推定、ピッチのトレースの実行有無を個別に切り替えることができます。
  • 編集中の状態を保存し、後でやり直すことができます。
  • メモリを大量に使用します。スペックの低いPCでのご利用は厳しいかもしれません。


  • 調整お手伝い version 1.17 更新 2014/12/21
  • 入力したパラメータに従い、エンベロープとMode2のポルタメントを自動設定します。
  • あくまで調声支援ツールです。聴き心地の良い調声にするためには、手作業での調整は手で調整は必須です。


その他
  • UTAU用合成エンジン bkh01 version 0.55 更新 2012/12/01
  • 元音声を声成分とノイズ成分を表現するパラメータに分解し、再合成する手法により実現しています。
  • 合成手法の特徴を生かした、独自の加工を行うフラグが追加されています。
  • 一部の既存のフラグの意味や利き方が変えられています。
  • 単音の音源のみ対応しています。合唱音源のような複数の音が混在している音源では使用できません。
  • 音源により品質に差が出ます。特に男声音源はノイズが多くなるようです。
  • アタックが強い音源は、アタック部分がノイズに化けて汚く聞こえることがあります。
  • 原音単位の中間ファイルを作成しませんが、初回実行時、bkh01.exeを配置したフォルダ内に「bkhnoise.dat」とい う中間ファイルが作成されます。
  • UTAU用合成エンジン tn_fnds version 0.0.9 更新 2013/4/11
  • このプログラムは森勢将雅氏のWORLD版UTAU合成エンジン『エターナルフォースブリサンプラージェントリー・ウィープス 　〜相手は死ぬ，俺も死ぬ〜』(EFB-GW) をカスタマイズし、連続音や子音速度、一部のフラグに対応させたものです。
  • 飴屋／菖蒲氏のworld4utauの ソースコードについても流用させていただいております。
  • 完全に他人の褌（ふんどし）です。森勢将雅氏、飴屋／菖蒲氏には頭が上がりません。
  • g,t,B,b,Wフラグに対応しているほか、A,O,eといった独自フラグを実装しています。（詳細はreadme.txtを 参照）
  • 高速化やメモリ使用量削減、フラグ対応のために、合成エンジンのコア部分に手を加えていますが、合成手法の基本的な考え方は変えてい ません。（変えていないつもりです。）
